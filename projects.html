imuStore DB

Database engine written in pure Python with zero dependencies. Every update creates a new immutable tree path; a commit is one atomic root swap, so readers never see a half-written state. Includes append-only storage, file locking, compaction, integrity audits and a full CLI.

GitHub

GAN Observatory

GAN Observatory is a real-time, in-browser studio for training and analyzing Generative Adversarial Networks. Watch a neural net learn a distribution live, steer training mid-run while a mode-collapse alarm watches, save and serve models over a REST API, and race two GANs in an A/B derby. Built with Python, FastAPI, WebSockets and NumPy, containerized with Docker and backed by 42 tests and CI.
