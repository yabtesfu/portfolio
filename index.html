imuStore DB

Database engine written in pure Python with zero dependencies. Every update creates a new immutable tree path; a commit is one atomic root swap, so readers never see a half-written state. Includes append-only storage, file locking, compaction, integrity audits and a full CLI.

GitHub
02

GAN Observatory

GAN Observatory is a real-time, in-browser studio for training and analyzing Generative Adversarial Networks. You watch a neural net learn a distribution live, steer training mid-run (learning rate, loss, stabilizers) while a mode-collapse alarm watches, save and serve models over a REST API, and race two GANs side by side in an A/B derby with a time-travel scrubber. Built with Python, FastAPI, WebSockets and NumPy, containerized with Docker and backed by 42 tests and CI.

GitHub

03

Kuas Meda

Kuas Meda is a full-stack football field booking and player-matching platform designed for Addis Ababa. It solves manual field reservations, unclear waiting lines, and the struggle to find enough players for a match. Players browse available fields, view open groups, apply to join matches, and manage upcoming sessions. Group organizers create sessions, accept or reject applicants, and manage team capacity. Field operators get a dashboard to add fields, create time slots, and manage daily schedules — with admin views, role-based access, and a foundation for payments, player ratings and smart matching.
