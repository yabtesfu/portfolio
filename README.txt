Overview
--------
A fast, responsive single-page portfolio built with plain HTML, CSS, and
JavaScript, with no framework and no build step. The site is organized into five
sections:

  Home        Centered hero with an animated signature logo, a single-run
              typing headline, and a live tech-stack strip.
  Projects    Featured engineering work with descriptions and GitHub/live links.
  About       Professional summary, full tech stack, and live typing and
              LeetCode statistics.
  Experience  Timeline of professional and academic experience.
  Contact     Direct email and social/professional links.

Features
--------
  - Light/dark theme toggle with localStorage persistence.
  - Animated cursive signature logo that replays on scroll-to-top.
  - Scroll-reveal animations, count-up statistics, and a live LeetCode donut.
  - Fully responsive layout for desktop, tablet, and mobile.
  - Accessible color contrast in both themes.
